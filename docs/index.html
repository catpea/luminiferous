A lightweight reactive templating system that combines ES6 template literals with reactive signals for dynamic DOM updates. Supports automatic subscription management and memory cleanup.

Example

// Basic usage
import { html, Signal } from "luminiferous";
const name = new Signal("World");
const element = html`<div>Hello ${name}!</div>`;
document.body.appendChild(element);

// Updates automatically reflect in DOM
name.set("Universe");

// Cleanup when done
element.unsubscribe();
